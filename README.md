# Sobre o projeto

O projeto foi desenvolvido durante a NLW#5, evento organizado pela rocketseat, se trata de uma  API chat de mensagem, para o atentende possa oferecer soport os cliente.

## TECNOLOGIAS USADAS  
        TypeScript
        Sqlite
        Express
        WebSocket
